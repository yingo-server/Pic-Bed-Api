随机图片 API · 从 8K 压缩库中随机选图

提供 PC / 手机 / 通用 三类图片的随机访问，支持返回 dataURL 或源链接。

请求参数

参数说明可选值默认
type 图片类型 pc / phone / general —
format 返回格式 dataurl / url —

💡 general 会从 PC 和手机图片库中随机选一个来源。

示例与交互

点击下方按钮可实时请求，或在新窗口查看返回结果。

1. PC · dataURL

GET /api/image?type=pc&format=dataurl

2. PC · 源链接

GET /api/image?type=pc&format=url

3. 手机 · dataURL

GET /api/image?type=phone&format=dataurl

4. 手机 · 源链接

GET /api/image?type=phone&format=url

5. 通用 · dataURL

GET /api/image?type=general&format=dataurl

6. 通用 · 源链接

GET /api/image?type=general&format=url

图片存放在 public/ppic/（电脑）和 public/mpic/（手机）中，随机返回。
